What are the default periods for observations and CMIP6 model evaluations delived with the package:
==============
The tool tires to cover as much as possible the last 25 years worth of fully available seasons (DJF,MAM,JJA,SON) within the CMIP6 framework. For model data that means the last December of 2014 is removed, and the final period is December 1989 till November 2014.
